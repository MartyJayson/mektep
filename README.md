## Requirements 
* `PHP`
* `Composer 1.10.1`
* `git 2.25.1`

## Steps
* `composer update`
* `create .env file`
* `add database configuration in .env as written in .env.example`
* `php migrate.php`
* cd into `public` directory
* `php -S localhost:8000`
